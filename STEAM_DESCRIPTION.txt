[h1]Moony's Magnificent Base[/h1]

[b]Moony's Magnificent Base[/b], or [b]MMB[/b], is the shared base mod for my other Moony's Magnificent mods.

By itself, it does not try to be flashy. It mostly contains shared helpers, reusable dialog systems, and common settings that the other mods use so I do not have to copy the same stuff into every project again and again.

[b]Features[/b]

[list]
[*]Shared helper functions.
[*]Reusable dynamic dialog framework for Zeus/module tools.
[*]Dialog import/export helpers.
[*]Shared ACE carry and drag weight settings.
[*]Debug logging settings for the Moony's Magnificent mod family.
[/list]

[b]Usage[/b]

Moony's Magnificent Computers, Modules, Additions, and future mods can depend on this instead of each carrying their own slightly different copy of the same helper code.

That is the whole idea. Not glamorous, but useful.

[b]Requirements[/b]

[list]
[*]CBA_A3
[*]ACE3
[/list]

You probably only need to load this because another one of my mods requires it.
